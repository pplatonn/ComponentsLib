## Типографика

### Размеры шрифтов
 
 Мелкий текст, подписи, метки 
 `--font-size-s` `0.8rem`
  Основной текст (по умолчанию) 
 `--font-size-m` `1.1rem`
  Заголовки, крупные элементы 
 `--font-size-l` `1.4rem`

### Скругления

 `--radius` `0.5rem` Скругление углов всех интерактивных элементов 

## Отступы

### Система отступов

 `--spacing-s` `0.3rem` Минимальные отступы 
 `--spacing-m` `1.1rem` Стандартные отступы 
 `--spacing-l` `2.1rem` Крупные отступы

## Примеры использования

### В компонентах Vue
```vue
<template>
 <button 
  class="ui-button" 
  :data-layout="layout"
  :data-disabled="isDisabled"
 >
  <slot />
 </button>
</template>

<style scoped>
.ui-button {
 padding: var(--spacing-m);
 font-size: var(--font-size-m);
 border-radius: var(--radius);
}

.ui-button[data-layout="primary"] {
 background-color: var(--color-primary);
 color: var(--color-white);
}

.ui-button[data-layout="primary"]:hover {
 background-color: var(--color-primary-hover);
}
</style>
